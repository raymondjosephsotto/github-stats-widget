# github-stats-widget
A React component that displays GitHub profile stats — plug in your own API endpoint

## Local preview

Run the demo app locally:

```bash
npm run dev
```

Then open the local Vite URL and point the form at your API endpoint.

Optional:

```bash
VITE_GITHUB_STATS_API_URL=https://your-api.vercel.app/api/github npm run dev
```
